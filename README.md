# Custom GPT Documentation

This repository contains system cards for custom GPTs developed for legal education at Penn Carey Law School.

## Available Custom GPTs

### 1. Textor: Reading Comprehension Quiz Generator
Generates text-anchored, advanced reading comprehension multiple-choice questions for legal academic texts.

**Access:**
- [ChatGPT EDU (Penn)](http://bit.ly/4o0VcZn)
- [ChatGPT (Paid, non-Penn)](https://bit.ly/4878vkQ)

[View System Card](textor-system-card.md)

### 2. Examiner: Complex MC Exam Questions Generator
Generates precise, text-based multiple-choice questions that test deep understanding of uploaded materials.

**Access:**
- [ChatGPT EDU (Penn)](https://bit.ly/3M8Nij4)
- [ChatGPT (Paid, non-Penn)](https://bit.ly/48bt7Zp)

[View System Card](examiner-system-card.md)

### 3. AI Hardener: Edit Essay Questions for AI Resistance
Evaluates and rewrites law school essay exam questions to make them significantly more resistant to generative AI tools.

**Access:**
- [ChatGPT EDU (Penn)](https://bit.ly/4ogm2gg)
- [ChatGPT (Paid, non-Penn)](https://bit.ly/49ZMiYu)

[View System Card](ai-hardener-system-card.md)

## About System Cards

Each system card includes:
- Overview and purpose
- Complete system prompt
- Core functionality details
- Requirements and constraints
- Output specifications
- Access links
